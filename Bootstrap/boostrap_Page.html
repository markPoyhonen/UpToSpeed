What is WebPleb?

Web pleb is a site intended for beginning users in web development to familiarize themselves with the recources and tools that are located on the web that help in real world applications. We are a non-profit site with the intention of helping out beginner web devolopers expand their knowledge of how to create and develop better sites for better users.

WebPleb is not a site that provides original work. We are simply a host site to centralize developer tools and sites that are intended to help in the process of development.
